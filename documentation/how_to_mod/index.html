Introduction

Ce guide vous accompagnera dans la création de mods pour R-Type en utilisant notre système de modules dynamiques. Vous apprendrez à créer, compiler et intégrer vos propres modifications au jeu.

Prérequis

  • CMake (version 3.10 minimum)
  • Un compilateur C++ compatible
  • Bibliothèques SFML installées
  • Connaissance de base en C++ et en programmation orientée objet

Problèmes Courants

  • Erreur de liaison : Vérifiez que toutes les dépendances sont correctement liées dans CMakeLists.txt
  • Symboles manquants : Assurez-vous que la fonction entryPoint() est déclarée avec extern "C"
  • Crash au chargement : Vérifiez la compatibilité des composants utilisés

Bonnes Pratiques

  • Testez toujours si les composants existent avant de les utiliser
  • Documentez clairement les dépendances de votre mod
  • Utilisez le système de gestion d’erreurs approprié
